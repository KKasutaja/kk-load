* kk-load

* This is an allowlisting system for FiveM servers.

* Requirements
  * [oxmysql](https://github.com/overextended/oxmysql)

## Installation
  * Drag 'kk-load' to your resources folder
  * Go find your server.cfg file
  * Start resources in this order
    * start mysql-async
    * start kk-load
  * Start your server.
